
# MamaCare Bot (Telegram)

Спокойный помощник для родителей: быстрый триаж (RED/AMBER/GREEN), без диагнозов и дозировок. Работает на Node.js + Telegraf + OpenAI.

## Быстрый старт

1) Установите Node.js 18+  
2) Распакуйте проект, откройте в VS Code  
3) Скопируйте `.env.example` → `.env` и заполните:
```
TG_BOT_TOKEN=ваш_токен_бота_от_BotFather
OPENAI_API_KEY=ваш_openai_api_key
OPENAI_MODEL=gpt-4o-mini
ADMIN_IDS=123456789   # через запятую, если несколько
TZ=Asia/Tashkent
```
4) Установка зависимостей и запуск:
```
npm install
npm run start
```
5) Напишите боту `/start` в Telegram.

### Что есть в MVP
- Формат ответа строго: [RED/AMBER/GREEN] → шаги сейчас → наблюдать → пояснение → напоминание → (уточняющие вопросы)
- Автодайджест в 19:00 по Asia/Tashkent в админ‑чат(ы)
- Лёгкая аналитика в `data/metrics.ndjson` (анонимно)

### Полезные команды/кнопки
- `/start` — приветствие и меню
- «FAQ», «RED признаки», «Язык» (RU/UZ)

### Замечания
- Проект использует **ESM** ("type": "module"). Если у вас старые примеры с `require(...)`, они не подойдут.
- Локальные логи и метрики пишутся в `data/`.
