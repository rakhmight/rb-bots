# MBI Bot (RU/UZ)
- Тест Maslach Burnout Inventory (22 вопроса), RU/UZ
- Профиль: Имя, Фамилия, Должность (обязателен перед стартом)
- Экспорт: JSON и Excel — только для админа
- Опционально: удалённый запуск для пользователя (админ)

## Старт
```bash
npm i
cp .env.example .env  # вставьте BOT_TOKEN и ADMIN_IDS
npm start
```
