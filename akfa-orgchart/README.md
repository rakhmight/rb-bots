# AKFA MEDLINE — Interactive Org Chart (v1.3.0)

Запуск:
```bash
npm i
npm run dev
```

- Логотип и ФИО в шапке настраиваются в `src/config.js`.
- В `src/orgData.js` вынесены отдельными карточками подразделы развития (маркетинг, сервис, проекты, международный, страхование).
- Анимация раскрытия: framer-motion. Иконки: lucide-react. Стили: Tailwind.

Сборка:
```bash
npm run build
npm run preview
```
