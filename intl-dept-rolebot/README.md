# International Department RoleBot

Бот для распределения задач и ежедневных чек-листов для International Department.

## Запуск

```bash
npm install
cp .env.example .env   # и вписать реальные значения
npm start
```

Телеграм-бот: назначение задач, ежедневные чек-листы, авто-перенос невыполненных задач на следующий день, экспорт отчётов в XLSX/PDF.
