# Support Hub Bot (Akfa Medline)

Единый Telegram-бот-хаб для перенаправления сотрудников в профильные боты техподдержки.

## Структура

- `src/index.js` — основной код бота.
- `.env` — переменные окружения (создайте из `.env.example`).

## Настройка и запуск

1. Установите зависимости:

   ```bash
   npm install
   ```

2. Создайте файл `.env` на основе `.env.example` и укажите токен:

   ```env
   BOT_TOKEN=ваш_токен_бота_от_BotFather
   ```

3. Запустите бота:

   ```bash
   npm start
   ```
