# Kelajack Leads Bot

Telegram-бот для записи клиентов в учебный центр **Kelajack** (RU/UZ/EN).
Собирает заявку: ФИО ребёнка → возраст → район → телефон, отправляет админу карточку, хранит статистику и умеет выгружать Excel.

## Установка
1) Node.js 18+
2) В корне проекта:
```bash
npm i
```

## Настройка
Скопируй `.env.example` → `.env` и заполни:
- `BOT_TOKEN`
- `ADMIN_IDS` (через запятую)
- `DB_PATH` (опционально)
- `BRAND` (опционально)

## Запуск
```bash
npm run start
```

## Команды
- `/start` — начать анкету
- `/myid` — узнать свой Telegram ID
- `/admin` — админ-панель (статистика и Excel)

## Примечание
Админы должны хотя бы 1 раз открыть бота, иначе Telegram может не позволить боту написать им первым.
